### 5. 前端渲染组件（React）

```tsx
// src/components/Pet.tsx
import { useEffect, useState } from 'react';
import { invoke } from '@tauri-apps/api/core';

export function Pet() {
  const [frame, setFrame] = useState<string>('');
  
  useEffect(() => {
    invoke('load_skin', { path: 'assets/skins/default' })
      .then(skin => {
        // 加载第一帧
        const framePath = skin.states.idle.frames[0];
        setFrame(framePath);
      });
  }, []);
  
  return (
    <div className="pet">
      {frame && <img src={frame} alt="pet" />}
    </div>
  );
}
```

**注意事项：**

1. 图片路径需要转换（本地文件 → 可访问 URL）
   - 方案 A：使用 `convertFileSrc` from `@tauri-apps/api/core`
   - 方案 B：返回 base64
   
2. Cargo.toml 修复后需要重新编译：
   ```bash
   cd src-tauri && cargo build
   ```

3. 资源文件需要在 `tauri.conf.json` 中配置：
   ```json
   "bundle": {
     "resources": ["assets/**"]
   }
   ```
