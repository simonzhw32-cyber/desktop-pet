## Goal

搭建 Tauri 2 桌宠应用脚手架，确保 build/lint/test 全绿，完成 README 与 DEVELOPMENT.md。

## Scope

**In scope:**
- `desktop-pet/` 根目录
- Tauri 2 项目初始化（前端选型：建议 React + TypeScript）
- `README.md` 项目说明
- `docs/DEVELOPMENT.md` 开发命令文档
- `.github/workflows/ci.yml` CI 骨架（build + lint + test）
- 基础目录结构

**Out of scope:**
- 桌宠核心功能实现（皮肤加载、状态机、透明窗口等）
- 皮肤编辑器
- 任何业务逻辑

## Acceptance criteria

- [ ] Tauri 2 项目初始化完成，`pnpm tauri dev` 可启动空白窗口
- [ ] `README.md` 包含项目介绍、技术栈、快速开始
- [ ] `docs/DEVELOPMENT.md` 包含实际开发命令（lint/test/build）
- [ ] `.github/workflows/ci.yml` CI 配置存在且语法正确
- [ ] `pnpm lint` 通过（或等效命令）
- [ ] `pnpm test` 通过（可以是空测试集）
- [ ] `pnpm build` 或 `pnpm tauri build` 成功
- [ ] Git 初始化完成，`.gitignore` 合理

## Commands（闸门）
